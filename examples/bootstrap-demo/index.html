alia/HTML & Bootstrap

This is a demonstration of alia/HTML's interface to Bootstrap components. Not all Bootstrap components currently have dedicated interfaces, but more will be added here as they're created.

This page is styled with the Bootswatch Lux theme, but nothing in the code is specific to that theme, so you should be free to use any Bootstrap-compatible theme. (alia/HTML currently targets Bootstrap 4.5.x.)
